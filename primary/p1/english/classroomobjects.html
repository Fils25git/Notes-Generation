UNIT 2: CLASSROOM OBJECTS



Lesson 1: What is this? What is that?

We ask questions to know the name of an object.

We use:

What is this? → for something near us.
What is that? → for something far from us.



Examples:

This is a book.
That is a chair.



Short dialogue:

Teacher: What is this?
Child: This is a pen.

Teacher: What is that?
Child: That is a desk.



Exercise:

1. Complete the sentences:
a) .......... is this? It is a bag.
b) .......... is that? It is a blackboard.

2. Point to a near object and say: This is a ______.
3. Point to a far object and say: That is a ______.


Lesson 2: Introducing colours – What colour is this?

We use colours to describe objects.

We ask:

What colour is this?
What colour is that?



Examples:

This book is red.
That chair is green.
This pen is blue.



  • red
  • blue
  • green
  • yellow
  • black
  • white


Short dialogue:

Teacher: What colour is this?
Child: It is yellow.

Teacher: What colour is that?
Child: It is black.



Exercise:

1. Write the colours:
a) The pen is ______.
b) The bag is ______.

2. Look around your classroom and list 3 coloured objects.
3. Ask your partner: What colour is this?


Lesson 3: Sounds and spellings

Letters have sounds. We repeat and practise spelling.



Examples:

b – bag
d – desk
c – chair
r – red



Exercise:

1. Say the sound of letter b.
2. Spell the word desk.
3. Match the letter with the object.
